# Ontario Mushroom Competitors: Growers and Grower-Distributors

Checked on April 12, 2026 using supplier public websites.

This file contains Ontario mushroom farms and grower-distributors that are more likely competitors than direct buyer leads for a shiitake and oyster mushroom farm.

## Type Legend

- `Grower / farm`: Primarily grows its own mushrooms and sells direct or through partners.
- `Grower / farm + distributor`: Grows mushrooms but also operates broader wholesale or distribution channels.

- If a public site did not explicitly list a mushroom type, it is not marked as verified here.
